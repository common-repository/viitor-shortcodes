== Description ==
Viitor shortcodes plugin is useful for dynamic button with attribute where it can be used through backend visual editor and it shown in frontend.
  
No coding Required,The plugin takes care of everything. 

Button shortcodes for ease of use.

It works with classic editors.

Create unlimited number of buttons.

Works with all modern browsers, degrades gracefully for others.

Predefined defaults make getting started super easy.

For hyperlink add attribute link to button shortcode. 

 For example: [ww_vcsc_boxes boxtype="boxstandard" link="https://viitorcloud.com/" target="_blank" showcontent="Click Me"][/ww_vcsc_boxes]
 
Supports wordpress visual editor to see your buttons in your editor!

Use the shortcode anywhere in your content and the button will appear.

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can go through wordpress post/page visual editor and you will see symbol to add shortcode.
